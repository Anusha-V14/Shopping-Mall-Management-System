# Shopping Mall Management System

## Description
A backend application built using Spring Boot to manage shopping mall operations. The project exposes REST APIs for customer management and integrates with PostgreSQL using Spring Data JPA.

## Features
- Add Customer
- Get Customer by ID
- Get All Customers
- Update Customer
- Delete Customer

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- REST APIs

## How to Run
1. Import project into Spring Tool Suite or Eclipse
2. Configure PostgreSQL database in application.properties
3. Run the main Spring Boot class
4. Test APIs using Postman
